# India AI Summit Explorer

An interactive web application to explore, categorize, and search across **363+ live stream sessions** from the IndiaAI Impact Summit 2026.

## Features

- **7 Topic Categories** — Governance, Infrastructure, Sectoral Innovation, Social Impact, Safety, Future Foundations, Keynotes
- **Real-time Search** — Filter across all sessions instantly
- **Video Playback** — Embedded YouTube player with session details
- **Transcript Search** — 32 sessions with searchable transcripts
- **AI Chat Assistant** — Ask questions sourced from transcripts and session titles
- **Favorites** — Bookmark sessions to watch later

> **Note:** This is not an official site. Content is sourced from the IndiaAI YouTube channel and impact.indiaai.gov.in.
